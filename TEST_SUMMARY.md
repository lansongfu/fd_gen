## 🔍 复杂场景验证

### Test 8: 星型拓扑 + 混合场景

**拓扑结构:**
```
        CPU
         |
    INTERCONNECT
    /         \
MEMORY     PERIPHERAL
```

**测试信号:**
- `ADDR` (大写，32 位) → FD 端口：`FD_ADDR_FROM_CPU`
- `DataBus` (混合，32 位) → FD 端口：`FD_DATABUS_FROM_CPU`
- `{req,ack}` (小写，拼接) → FD 端口：`fd_req_from_cpu`
- `status` (双向，2 位) → FD 端口：`fd_status_from_peripheral` + `<->`

**验证结果:**
- ✅ 星型拓扑正确路由
- ✅ 大小写规则正确应用
- ✅ 拼接信号正确拆分
- ✅ 双向信号正确处理
- ✅ 位宽不一致 WARNING
